Web Application

D3M is a collaboration solution built to help technicians, project managers, sales reps, engineers, design network topology, rack and floor plan diagrams all while auto-generating project inventory, quotes, proposals and documentation.

This web application help its users to reduce repetitive tasks while keeping projects and team in sync. D3M automatically create quotes, proposals and equipment lists from the diagrams users design.
