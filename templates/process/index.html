Process



Discovery


Define the problem - that needs a solution. Kick-off meeting, requirements, specifications.



Research


User research, personas, surveys, interviews, ethnography, contextual research, competitive analysis.



Architecture


Synthesize information, build share understanding - journey maps, mental models, user stories. Affinity diagram, taxonomy, card sorting.



Prototyping and Testing


Sketch multiple solutions, get early feedback and refine. Wireframes. Build interactive prototypes. A/B testing. Create visual designs - concepts, stylescapes, design systems.



Validate and Iterate


Collect feedback - Evaluate test results and determine necessary modifications.
